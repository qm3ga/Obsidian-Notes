## Структура

### Общая структура

+ На ресурсе реализована интуитивная навигация
+ Основной контент представлен в виде стандартной сетки проектов
+ Есть фильтры по типу проекта, креативным областям, инструментам и цветовой палитре
+ Предусмотрен поиск по проектам, авторам и изображениям
+ Глубина композиции — 2–3 уровня в зависимости от страницы
+ Есть галерея проектов и раздел с вакансиями

### Страница работы

+ Минималистичный дизайн, который не отвлекает от просмотра проекта
+ Изображения представлены в виде вертикальной ленты
+ Предусмотрена система лайков и комментариев
+ Авторство проекта могут разделять несколько человек
+ Есть возможность указать инструменты и программное обеспечение
+ После просмотра проекта пользователю предлагаются другие работы автора и похожие проекты

### Страница с вакансиями

+ Есть сортировка по типу занятости, креативным областям и локации
+ На странице вакансии отображается информация о формате работы (офис / удалённо) и зарплате
+ Есть возможность ознакомиться с компанией через внешние ресурсы или официальный сайт

### Дополнительно

+ Есть обучающие материалы и курсы для дизайнеров разного уровня
+ Проводятся вебинары и онлайн-мероприятия
+ Часть образовательного контента предоставляется компанией Adobe
