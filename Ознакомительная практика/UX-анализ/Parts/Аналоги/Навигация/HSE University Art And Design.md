## 1. Удобство переходов

- Навигация построена по принципу многоуровневого меню с переходами между разделами
- Каждый переход открывает новую страницу с полной перезагрузкой контента
- На страницах с портфолио используются превью в виде карточек, кликабельных для перехода к детальному описанию
- В разделе новостей и событий реализована навигация по датам
- На длинных страницах с описанием программ используются якорные ссылки для быстрого перехода к разделам внутри страницы
- На длинных страницах слева внизу присутствует кнопка для быстрого возврата в начало

## 2. Понятность меню

- Содержит основные разделы: "О школе", "Поступающим", "Студентам", "Портфолио", "Новости" и другие
- В некоторых разделах используется дополнительное меню для навигации по подразделам
- Текущий раздел выделяется в меню подчеркиванием
- В правом верхнем углу расположен переключатель языка (русский/английский)
- Иконка поиска находится в верхней панели, открывая строку поиска по сайту при клике
- В нижней части страницы дублируются основные разделы и добавлены контактные данные, ссылки на социальные сети и другая информация

## 3. Наличие поиска

- Поиск вызывается кликом на иконку лупы в правом верхнем углу шапки сайта
- После клика открывается выпадающая строка для ввода поискового запроса по всему сайту
- Результаты выводятся на отдельной странице в виде списка с заголовками и краткими описаниями
- Поиск охватывает все разделы сайта
- Расширенные фильтры для уточнения результатов поиска не предусмотрены
- В результатах поиска ключевые слова выделяются жирным для удобства восприятия
- Результаты поиска выводятся в порядке релевантности без возможности изменения сортировки
