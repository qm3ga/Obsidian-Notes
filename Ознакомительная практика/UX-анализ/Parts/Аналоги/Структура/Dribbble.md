## Структура

### Общая структура

+ Основной контент на сайте — карточки работ (shots), расположенные в стандартной сетке
+ Есть категории и тематические подборки работ
+ Предусмотрены фильтры по цветам, тегам и времени публикации
+ Есть поиск по дизайнерам, работам и услугам
+ Есть система лайков и сохранения работ

### Страница работы

+ Работа отображается в виде карточки с изображением или анимацией
+ Автор может добавить описание проекта и используемые инструменты
+ Отображается профиль автора и ссылки на его портфолио
+ Есть возможность поставить лайк, оставить комментарий или сохранить работу

### Страница услуг

+ Более структурированная композиция, позволяющая работодателю быстро получить информацию о специалисте
+ Отображаются навыки, специализация и примеры работ
+ Работодатель может нанять специалиста или запросить персональную услугу

### Дополнительно

+ Есть блоги и статьи о дизайне
+ Проводятся дизайнерские челленджи и конкурсы
+ Есть раздел с вакансиями
+ Работает служба поддержки пользователей
