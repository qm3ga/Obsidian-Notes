## Структура

### Общая структура

+ Основной контент на сайте — карточки работ художников, расположенные в сетке
+ Есть разделы с популярными, новыми и трендовыми работами
+ Доступны категории работ (3D, концепт-арт, иллюстрация, персонажи, окружение)
+ Предусмотрены фильтры по тегам, программам, популярности и времени публикации
+ Есть поиск по художникам, студиям, работам и обучающим материалам
+ Есть система лайков, комментариев и сохранения работ

### Страница работы

+ Галерея проекта с возможностью размещения изображений, видео и 3D-просмотра моделей
+ Есть описание проекта, используемые инструменты и этапы создания
+ Отображается информация об авторе и ссылки на его профиль
+ Есть возможность поставить лайк, оставить комментарий или сохранить работу

### Страница художника

+ Профиль художника с аватаром, баннером и описанием
+ Сетка всех работ автора
+ Указаны навыки, специализация и используемые программы
+ Есть возможность подписаться на художника
+ Доступна контактная информация и ссылки на соцсети

### Marketplace

+ Площадка для продажи цифровых ресурсов
+ Можно продавать и покупать:
  + 3D-модели
  + текстуры
  + ассеты
  + обучающие материалы

### Вакансии

+ Раздел вакансий для художников и дизайнеров
+ Студии могут публиковать предложения работы
+ Художники могут откликаться и отправлять портфолио

### Дополнительно

+ Обучающая платформа с уроками и курсами
+ Челленджи и конкурсы для художников
+ Подписки и персональная лента работ
+ Сообщество и комментарии
+ Служба поддержки
