# Сравнительный анализ аналогов

| Критерий                              | Behance                                                                   | ArtStation                                                               | Dribbble                                                        | HSE (ВШЭ)                                                      |
| ------------------------------------- | ------------------------------------------------------------------------- | ------------------------------------------------------------------------ | --------------------------------------------------------------- | -------------------------------------------------------------- |
| Цель платформы                        | Универсальная платформа для публикации и продвижения портфолио            | Портфолио для специалистов игровой и 3D-индустрии                        | Площадка для демонстрации дизайнерских решений (UI/UX, графика) | Демонстрация студенческих работ в образовательном контексте    |
| Целевая аудитория                     | Дизайнеры, иллюстраторы, работодатели                                     | Художники, 3D-специалисты, гейм-дизайнеры                                | UI/UX дизайнеры, графические дизайнеры                          | Студенты, абитуриенты, преподаватели                           |
| Формат представления работ            | Полноценные проекты (кейсы) с текстом и визуалом                          | Детализированные кейсы с акцентом на визуальную составляющую             | Короткие “шоты” (отдельные фрагменты работ)                     | Проекты студентов, часто с привязкой к курсам                  |
| Структура контента                    | Гибкая структура проектов, возможность добавления описания, этапов работы | Структура ориентирована на демонстрацию финального результата и процесса | Минимальная структура, упор на быстрый просмотр                 | Иерархическая структура (по курсам, направлениям, дисциплинам) |
| Навигация                             | Развитая система фильтров, категорий и поиска                             | Навигация по направлениям и специализациям                               | Простая навигация через ленту и теги                            | Навигация через образовательную структуру (курсы, программы)   |
| Функциональные возможности            | Публикация проектов, лайки, комментарии, подписки                         | Публикация работ, участие в конкурсах, профессиональное сообщество       | Публикация работ, лайки, подписки                               | Просмотр работ, базовая навигация, ограниченное взаимодействие |
| Уровень взаимодействия (социальность) | Высокий (социальная сеть)                                                 | Средний (профессиональное сообщество)                                    | Средний (лайки, подписки)                                       | Низкий (в основном просмотр)                                   |
| Удобство для размещения работ         | Высокое, но требует оформления кейсов                                     | Высокое для визуальных проектов                                          | Очень простое, но ограниченное по содержанию                    | Ограниченное (зависит от структуры сайта)                      |
| Удобство для просмотра работ          | Может быть перегружено из-за большого объёма контента                     | Удобно для визуального просмотра                                         | Быстрое и лёгкое потребление контента                           | Удобно для понимания учебного процесса, но менее гибко         |
| Сильные стороны                       | Универсальность, популярность, развитые инструменты                       | Глубина проработки проектов, профессиональная направленность             | Простота и скорость восприятия                                  | Связь с образовательным процессом, структурированность         |
| Слабые стороны                        | Перегруженность, отсутствие образовательного контекста                    | Узкая специализация                                                      | Поверхностность представления работ                             | Ограниченный функционал, слабая интерактивность                |
| Релевантность для проекта             | Полезна как пример структуры портфолио и соцфункций                       | Полезна для демонстрации глубины проектов                                | Полезна для UI и простоты взаимодействия                        | Наиболее близка по концепции (образовательная платформа)       |
| Что можно заимствовать                | Структуру кейсов, систему публикации, фильтры                             | Детализацию проектов, визуальный акцент                                  | Простоту интерфейса, быстрый просмотр                           | Иерархию по дисциплинам, образовательную логику                |
| Что не подходит                       | Избыточная социальность и перегруженность                                 | Узкая специализация под индустрию                                        | Недостаточная глубина контента                                  | Ограниченность функций и слабое взаимодействие                 |
| Вывод для разработки                  | Использовать как основу для структуры портфолио, упростив интерфейс       | Использовать для глубокой проработки проектов                            | Использовать для упрощения UX                                   | Использовать как основу архитектуры системы                    |
