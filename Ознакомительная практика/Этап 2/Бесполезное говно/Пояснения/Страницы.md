# Описываются по следующему шаблону:
## Название страницы

**Назначение страницы:**  
Кратко — зачем она нужна пользователю.



**Структура страницы включает следующие основные блоки:**

1. **Шапка (Header)**
    - логотип
    - меню навигации
    - поиск (если есть)
2. **Основной контент (Main)**
    - (перечисляешь ключевые блоки)
    - в порядке сверху вниз
3. **Дополнительные элементы**
    - фильтры / похожие / кнопки
4. **Подвал (Footer)**
    - контакты
    - ссылки



**Иерархия элементов:**

- что самое важное
- что второстепенное



**Логика расположения:**  
Почему блоки расположены именно так (с точки зрения UX).

# Распределение страниц



# Участник 1 — Вход и ориентация



## Страница 1. Главная страница

**Роль:** точка входа  
Содержит:

- описание платформы
- избранные работы
- быстрые переходы



## Страница 2. О кафедре

**Роль:** информационная страница
Содержит:

- описание направления
- преподавателей
- особенности обучения



## Страница 3. Контакты

**Роль:** связь  
Содержит:

- контакты
- соцсети
- форма связи



# Участник 2 — Навигация и поиск

## Страница 4. Каталог работ

**Роль:** основной список  
Содержит:

- карточки работ
- фильтры
- сортировку



## Страница 5. Результаты поиска

**Роль:** отдельный экран поиска  
Содержит:

- список найденных работ
- строку поиска
- фильтры



## Страница 6. Подборки работ

**Роль:** curated-контент  
Содержит:

- «Лучшие работы»
- «Рекомендуемые»



# Участник 3 — Просмотр контента

## Страница 7. Страница проекта

**Роль:** основной просмотр  
Содержит:

- галерею
- описание
- информацию



## Страница 8. Похожие работы

**Роль:** продолжение просмотра  
Содержит:

- список похожих проектов



## Страница 9. Страница автора

**Роль:** отдельная страница с информацией об авторе
Содержит:

- аватар / имя
- краткую информацию
- список работ
- фильтры по работам



# Участник 4 — Пользователь

## Страница 10. Профиль студента

**Роль:** личный кабинет  
Содержит:

- информация
- список работ



## Страница 11. Добавление работы

**Роль:** форма  
Содержит:

- поля
- загрузку



## Страница 12. Авторизация / регистрация

**Роль:** вход  
Содержит:

- логин
- регистрацию
