## Структура

### Общая структура

+ На ресурсе интуитивная навигация
+ Стандартная сетка работ с фильтром по типу работы, местоположению, использованным ресурсам и цветовой палитре
+ Предусмотрен поиск по проектам, людям, ассетам и изображениям
+ Глубина композиции - 2-3 уровня, в зависимости от страницы
+ Есть как галерея, так и список вакансий

### Страница работы

+ Минималистичный дизайн, ничего не отвлекает от работы автора
+ Изображения представлены в виде ленты
+ Предусмотрена система лайков и комментариев
+ Авторство могут разделять несколько человек
+ Есть возможность указать инструменты, использованные для выполнения работы
+ После просмотра всех изображений пользователю предлагают другие работы автора

### Страница с вакансиями

+ Есть сортировка по категориям занятости, областям деятельности и локации
+ На странице вакансии есть информация о необходимости ездить в офис и о зарплате
+ Есть возможность ознакомиться с компанией, но на сторонних ресурсах

### Дополнительно

На Behance есть возможность пройти курсы как для повышения квалификации, так и для новичков.

Есть страница с вебинарами, по больший части от Adobe.
