## Структура

### Общая структура

+ Основной контент на сайте - карточки работ, стандартная сетка
+ Легкодоступные категории работ
+ Предусмотрена фильтры по цветам, тегам, и времени публикации работы
+ Есть поиск по услугам, дизайнерам и работам
+ Есть система лайков
### Страница работы

+ Гибкая сетка, автор работы может выставить изображения по своему желанию
+ Доступная контактная информация
+ Есть возможность написать автору через встроенный мессенджер
+ Есть вкладка с обратной связью о работе

### Страница услуг

+ Более строгая композиция (по сравнению с карточкой работы), что способствует работодателю быстрее получить информацию о специалисте
+ Для работодателя есть возможность как нанять специалиста, так и запросить у него персональную услугу

### Дополнительно

Есть блоги профессиональных дизайнеров, креативные соревнования и служба поддержки
