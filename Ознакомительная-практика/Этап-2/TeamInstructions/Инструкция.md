# Задачи на этап


## 1. Общие



### 1.1. Карта сайта ([Site Map](SiteMap.md))

Что это:

- схема всех страниц
- иерархия
- связи

Что должно быть:

- все страницы
- не более 3 уровней вложенности
- логичная структура



### 1.2. Обоснование структуры сайта (текст)

Нужно написать:

- почему выбрана именно такая структура
- как она решает задачи пользователей
- на что опирались (аналогы + этап 1)



### 1.3. Модель навигации

Нужно описать:

- тип меню
- как пользователь перемещается
- есть ли:
    - поиск
    - фильтры
    - [хлебные крошки](ХлебныеКрошки.md)



## 2. Индивидуальные


### 2.1. Описание своих [страниц](Страницы.md) (3 на человека)

Для КАЖДОЙ страницы:

Нужно описать:

- какие блоки есть
- что главное
- как расположены элементы



### 2.2. [User Flow](UserFlow.md) (1–2 на человека)

Что это:

- сценарии пользователя (схемы)

Что должно быть:

- начало (вход)
- действия
- выбор (если есть)
- результат




### 2.3. Логика переходов

Нужно объяснить:

- откуда пользователь приходит
- куда уходит
- почему так удобно



### 2.4. Обоснование решений

Нужно написать:

- почему сделано именно так
- какие решения взяты из аналогов
