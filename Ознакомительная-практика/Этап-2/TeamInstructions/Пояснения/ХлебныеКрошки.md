# Хлебные крошки (Breadcrumbs)

## Что это

**Хлебные крошки** — это элемент навигации, который показывает где сейчас находится пользователь внутри сайта



## Пример

Главная → Каталог → 3 курс → UI-дизайн → Работа



## Зачем нужны

- помогает понять, где ты находишься
- позволяет быстро вернуться назад
- улучшает навигацию



## Где используются

- на странице проекта
- в каталоге
- в профиле
