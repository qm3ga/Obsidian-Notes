## Структура

#### Общая структура

+ Основной контент представлен в виде карточек студенческих проектов, расположенных в сетке
+ Есть разделение проектов по направлениям обучения (графический дизайн, анимация, иллюстрация, гейм-дизайн и др.)
+ Предусмотрена фильтрация по направлениям, курсам и типам проектов
+ Есть поиск по проектам и авторам
+ Проекты могут быть объединены в подборки или выставки

#### Страница проекта

+ Проект представлен в виде вертикальной ленты изображений и медиа
+ Есть описание проекта, его концепции и задач
+ Указывается автор проекта и направление обучения
+ Иногда отображается информация о преподавателе или учебном курсе
+ После просмотра проекта пользователю предлагаются другие студенческие работы

#### Страница автора

+ Профиль студента с именем и направлением обучения
+ Представлена галерея работ студента
+ Отображаются учебные проекты и дипломные работы
+ Есть возможность перейти к другим проектам того же автора

#### Дополнительно

+ Есть онлайн-галереи лучших студенческих работ
+ Публикуются проекты дипломных и курсовых работ
+ Платформа используется для демонстрации работ на выставках и образовательных мероприятиях
+ Есть интеграция с новостями и событиями школы дизайна
