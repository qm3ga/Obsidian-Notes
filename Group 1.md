# Экраны для генерации

## 1. Главная страница (high ≈ 2400px)

Блоки сверху вниз:

1. Шапка (см. общую структуру).
2. Hero-секция (height 480px):
   - Заголовок крупно: «Работы студентов кафедры компьютерного дизайна»
   - Подзаголовок: «Цифровое пространство для просмотра проектов направления РТУ МИРЭА»
   - Две кнопки: «Смотреть каталог» (primary), «О кафедре» (secondary)
   - Справа — большое изображение-плейсхолдер 600×400
3. Витрина избранных работ (height 720px):
   - Заголовок секции: «Избранные работы»
   - Сетка 3×3, карточки 400×220
   - Карточка: изображение-плейсхолдер 400×160, под ним название работы и имя автора
4. Блок подборок (height 360px):
   - Заголовок секции: «Тематические подборки»
   - Горизонтальная лента 3 карточек 400×280
   - Карточка подборки: изображение-плейсхолдер, название подборки, количество работ
5. Тизер «О кафедре» (height 320px):
   - Слева — изображение-плейсхолдер 560×240
   - Справа — заголовок «О направлении», абзац текста-плейсхолдера (3 строки), кнопка «Подробнее»
6. Подвал (см. общую структуру).

## 2. Каталог работ (high ≈ 1800px)

Блоки сверху вниз:

1. Шапка.
2. Заголовок страницы и счётчик (height 80px): «Каталог работ» слева, «Найдено: 247 работ» справа.
3. Двухколоночная область:
   - Левая колонка (width 240px, sticky): панель фильтров
     - Группа «Курс» — 4 чекбокса (1, 2, 3, 4 курс)
     - Группа «Дисциплина» — 6 чекбоксов (Графический дизайн, 3D, Веб-дизайн, UX/UI, Иллюстрация, Анимация)
     - Группа «Год» — 4 чекбокса (2026, 2025, 2024, 2023)
     - Группа «Тип проекта» — 3 чекбокса (Учебный, Курсовой, ВКР)
     - Кнопка «Сбросить фильтры» (secondary, full width)
   - Правая колонка (width 1000px):
     - Переключатели сортировки сверху (height 40px): «По дате», «По популярности», «По названию»
     - Сетка карточек 3×N, карточка 320×260
     - Карточка: изображение 320×200, ниже название работы, имя автора, тег дисциплины
4. Подвал.

## 3. Страница проекта (high ≈ 2200px)

Блоки сверху вниз:

1. Шапка.
2. Хлебные крошки (height 40px): «Главная / Каталог работ / [Название работы]».
3. Полноширинная галерея (height 720px): большое изображение-плейсхолдер 1280×720, под ним полоса миниатюр (4 штуки 200×120).
4. Двухколоночная область (height 480px):
   - Левая колонка (width 840px):
     - Заголовок проекта (крупный текст)
     - Имя автора с аватаром 40×40 рядом
     - Абзац описания концепции (5 строк текста-плейсхолдера)
   - Правая колонка (width 400px):
     - Блок мета-данных: «Дисциплина: Графический дизайн», «Курс: 2 курс», «Год: 2025», «Инструменты: Figma, Photoshop»
     - Теги (5 штук в виде закруглённых плашек)
     - Кнопка «Связаться со студентом» (primary, full width)
5. Блок похожих работ (height 360px): заголовок «Похожие работы», сетка 4 карточек 280×240.
6. Подвал.

## 4. Страница подборки (high ≈ 1800px)

Блоки сверху вниз:

1. Шапка.
2. Хлебные крошки (height 40px): «Главная / Подборки / [Название подборки]».
3. Шапка подборки (height 200px):
   - Заголовок крупно: «Лучшие работы по 3D-моделированию 2025»
   - Описание подборки (3 строки)
   - Строка с автором подборки: аватар 32×32 + «Составитель: преподаватель Иванов И.И.»
   - 4 тега
4. Панель действий (height 60px): счётчик «12 работ», кнопка «Добавить все в избранное» справа.
5. Сетка карточек работ 3×N, карточка 400×320:
   - Изображение-плейсхолдер 400×240
   - Название работы, имя автора
   - Иконка-сердце (лайк) в углу карточки
6. Блок похожих подборок (height 320px): заголовок «Похожие подборки», горизонтальная лента 3 карточек 400×280.
7. Подвал.
