# Задача: wireframe-макеты для учебного проекта

Нужно сгенерировать SVG-wireframe-макеты для 10 экранов веб-платформы. Это учебная работа для ознакомительной практики РТУ МИРЭА, направление «Компьютерный дизайн», 1 курс. Проект — онлайн-пространство для просмотра работ студентов кафедры.

## Что нужно

Только wireframe (низкая точность, Lo-Fi). НЕ нужен:
- UI-дизайн (цвета, типографика-кит, иконки кроме плейсхолдеров)
- Mid-fi и Hi-fi
- Интерактивность
- Mobile-версия (только desktop 1440px)
- Описания, комментарии, объяснения

## Технические требования к SVG

КРИТИЧНО: SVG должен открываться в Figma через File → Import как редактируемые слои.

1. Все стили — inline-атрибутами (`stroke="#666"`, `fill="none"`), НЕ через `<style>` блок и НЕ через классы.
2. Шрифт — только `font-family="Inter, Arial, sans-serif"`. Никаких кастомных шрифтов.
3. Структура слоёв через `<g>` с осмысленными `id` и `inkscape:label` (или комментарием). Пример: `<g id="header"> ... </g>`. Это нужно, чтобы при импорте в Figma слои были подписаны.
4. Каждый экран — отдельный `<svg>` блок viewBox 1440×N (высота под контент).
5. Все размеры в px, без процентов.
6. Текст-плейсхолдеры — `<text>` элементами с реальным русским текстом по спецификации ниже (НЕ Lorem Ipsum).
7. Изображения-плейсхолдеры — прямоугольник с диагональным крестом внутри (две `<line>`).
8. Кнопки — прямоугольник со скруглением `rx="6"` + текст по центру.
9. Поля ввода — прямоугольник с тонкой обводкой `stroke-width="1"` + плейсхолдер-текст.

## Палитра wireframe (только оттенки серого)

- Фон страницы: `#FFFFFF`
- Обводки и разделители: `#CCCCCC`
- Плейсхолдеры изображений: `stroke="#999999"`, `fill="#F0F0F0"`
- Кнопки primary: `stroke="#666666"`, `fill="#666666"`, текст `#FFFFFF`
- Кнопки secondary: `stroke="#666666"`, `fill="#FFFFFF"`, текст `#666666`
- Основной текст: `#1A1A1A`
- Вторичный текст и подписи: `#666666`
- Плейсхолдер-текст в полях: `#999999`

## Сетка

- Ширина страницы: 1440px
- Контентная область: 1280px (margin 80px слева и справа)
- Базовый отступ: 8px (все размеры кратны 8)
- Шапка: высота 72px
- Подвал: высота 240px

## Структура шапки (одинакова для всех 10 экранов)

```
[Логотип 120×32] [Каталог работ] [Профили] [Подборки] [О кафедре]    [Поиск 280×40] [Войти 100×40]
```

## Структура подвала (одинакова для всех 10 экранов)

```
Колонка 1: Логотип + краткое описание платформы
Колонка 2: Разделы (Каталог, Профили, Подборки)
Колонка 3: О кафедре (О направлении, Преподаватели, Контакты)
Колонка 4: Контакты (email, телефон, адрес)
---
© 2026 Кафедра компьютерного дизайна РТУ МИРЭА
```

## Формат ответа

Для каждого экрана:

1. Краткий заголовок-комментарий (название экрана, размеры).
2. Один SVG-блок целиком.
3. Никаких объяснений после SVG.
